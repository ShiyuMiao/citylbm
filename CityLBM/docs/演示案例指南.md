# CityLBM 演示案例指南

## 🎯 目标

通过一个简单的单体建筑风环境模拟，完整演示 CityLBM 插件的工作流程。

---

## 📋 前置条件

### 软件环境
- ✅ Rhino 7 已安装
- ✅ Grasshopper 已启用
- ✅ CityLBM 插件已加载（CityLBM.gha）

### 可选（完整模拟需要）
- ⚪ FluidX3D 已安装并配置

---

## 🚀 快速开始（5分钟演示）

### 第一步：创建建筑物几何

1. **打开 Rhino 7**
2. **创建一个简单的建筑物**：
   - 命令：`Box`
   - 角点：`(0, 0, 0)`
   - 尺寸：`10m x 10m x 20m`（长x宽x高）

3. **转换为 Mesh**：
   - 选中 Box
   - 命令：`Mesh`
   - 选择：`Coarse` 或 `Medium`

---

### 第二步：打开 Grasshopper

1. **启动 Grasshopper**：
   - Rhino 命令行输入：`Grasshopper`
   - 按 Enter

2. **检查 CityLBM 组件**：
   - 在组件面板中找到 **CityLBM** 标签页
   - 应该看到 3 个子标签：
     - **Scene**（场景管理）
     - **Simulation**（模拟）
     - **Results**（结果）

---

### 第三步：构建场景

**组件连接顺序**：

```
Rhino Box → Mesh → Create Scene → Add Buildings → Generate Grid → Run Simulation
```

#### 详细步骤：

**1. 添加 Create Scene 组件**
- 位置：CityLBM → Scene → Create Scene
- 设置：
  - Name: `"DemoBuilding"`
  - Wind Direction: `(1, 0, 0)` （X 方向风）
  - Wind Speed: `5` m/s

**2. 添加 Add Buildings 组件**
- 位置：CityLBM → Scene → Add Buildings
- 输入：
  - Scene: 连接 Create Scene 输出
  - Buildings: 连接 Rhino 中的 Mesh

**3. 添加 Scene Info 组件**（可选）
- 位置：CityLBM → Scene → Scene Info
- 用于查看场景统计信息

---

### 第四步：生成计算网格

**4. 添加 Generate Grid 组件**
- 位置：CityLBM → Simulation → Generate Grid
- 输入：
  - Scene: 连接 Add Buildings 输出
  - Cell Size: `1.0`（1米网格）

**输出信息**：
```
网格尺寸: 14 x 14 x 24
总单元格: 4704
流体单元: 4200
障碍物单元: 504
```

---

### 第五步：导出 Case 文件

**5. 添加 Run Simulation 组件**
- 位置：CityLBM → Simulation → Run Simulation
- 输入：
  - Scene: 连接 Add Buildings 输出
  - Grid: 连接 Generate Grid 输出
  - FluidX3D Path: （可选，填写 FluidX3D 安装路径）
  - Viscosity: `1.5e-5`（空气粘度）
  - Time Steps: `1000`
  - Save Interval: `100`

**输出**：
- Case Dir: `C:\Users\...\AppData\Local\Temp\CityLBM\DemoBuilding\`
- Success: `True`
- Message: Case 文件已生成

---

### 第六步：查看生成的文件

打开 Case 目录，应该看到以下文件：

```
DemoBuilding/
├── buildings.stl        # 建筑物几何文件
├── defines.hpp          # 配置文件
├── setup.cpp            # 场景设置
└── run_simulation.bat   # 运行脚本
```

## 📊 测试可视化功能（无需 FluidX3D）

如果你还没有安装 FluidX3D，可以使用示例 VTK 文件测试可视化功能。

### 步骤：

**1. 找到示例 VTK 文件**
```
CityLBM\examples\sample_output\result_100.vtk
```

**2. 添加 Read VTK 组件**
- 位置：CityLBM → Results → Read VTK
- 输入：
  - VTK Path: `C:\...\CityLBM\examples\sample_output\result_100.vtk`
  - Time Step: `-1`（读取所有）

**3. 查看输出**
- Points: 125 个网格点
- Velocity: 速度向量列表
- Pressure: 压力值列表

**4. 可视化速度场**
- 添加 Visualize Velocity 组件
- 连接 Read VTK 的输出

## 🔧 完整模拟流程（需要 FluidX3D）

### 1. 配置 FluidX3D 路径

在 Run Simulation 组件中填写 FluidX3D 安装路径：
```
C:\Path\To\FluidX3D
```

### 2. 运行模拟

- 双击 `run_simulation.bat`
- 或在 Grasshopper 中点击运行

### 3. 等待模拟完成

模拟时间取决于网格规模和硬件性能。

### 4. 读取结果

- 使用 Read VTK 组件读取 `output/` 目录中的 VTK 文件
- 使用可视化组件分析结果

## 📈 参数建议

### 网格尺寸（Cell Size）

| 场景类型 | 建议值 | 说明 |
|---------|--------|------|
| 快速测试 | 2.0 - 5.0 m | 粗糙网格，快速验证流程 |
| 初步分析 | 1.0 - 2.0 m | 平衡精度和速度 |
| 详细分析 | 0.5 - 1.0 m | 较高精度 |
| 精细模拟 | 0.25 - 0.5 m | 高精度，需要更多计算资源 |

### 时间步数（Time Steps）

| 目的 | 建议值 | 说明 |
|-----|--------|------|
| 流动初始化 | 500 - 1000 | 让流动充分发展 |
| 统计平均 | 2000 - 5000 | 需要足够的样本 |
| 瞬态分析 | 5000+ | 捕捉时间变化特征 |

## ⚠️ 常见问题

### 1. "场景中没有建筑物"
- 确保 Rhino 中的几何体已转换为 Mesh
- 确保 Add Buildings 组件正确连接

### 2. "网格对象无效"
- 检查 Generate Grid 是否成功运行
- 查看网格统计信息是否有异常

### 3. VTK 文件读取失败
- 确认文件路径正确
- 检查 VTK 文件格式是否为 ASCII

### 4. CityLBM 标签页未出现
- 重启 Rhino 和 Grasshopper
- 检查插件文件是否在 Grasshopper 库目录
